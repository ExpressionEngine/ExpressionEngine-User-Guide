# make:model

Model Generator -- Creates a new model for an add-on

## Options list:

```
    --addon=<value>
    -a <value>
        Folder for third-party add-on you want to add model to
```

## Examples:

### Generating a model for an add-on:

`php eecli.php make:model MyAwesomeModel --addon=my_existing_addon`
