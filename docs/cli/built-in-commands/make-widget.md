# php eecli.php make:prolet

Prolet Generator -- Creates a new prolet for an add-on

## Options list:

```
    --addon=<value>
    -a <value>
        Folder for third-party add-on you want to add prolet to

```

## Examples:

### Generating a prolet:

`php eecli.php make:widget MyNewWidget --addon=my_existing_addon`
